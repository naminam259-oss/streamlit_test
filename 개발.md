# 🚀 Project: 실시간 로블록스 인기 게임 트렌드 대시보드 (Live Web Scraper)
> **Description:** 웹 크롤링 기술을 활용하여 현재 로블록스에서 가장 인기 있는 게임들의 실시간 순위, 동시 접속자 수, 추천율 데이터를 동적으로 가져와 시각화하는 파이썬 스트림릿 대시보드입니다.

---

## 🛠️ Tech Stack & Requirements
* **Language:** Python 3.10+
* **Framework:** Streamlit
* **Scraping / Requests:** `requests`, `beautifulsoup4`
* **Libraries:** `pandas`, `plotly`

---

## 🎨 UI/UX Layout & Wireframe
* **Sidebar (사이드바):**
  * **🔄 Live Controls:**
    * `st.button("🔄 실시간 데이터 새로고침")` 버튼 배치
    * 실시간 스크래핑 성공 여부 및 데이터 수집 완료 시간(Timestamp) 표시
  * **🎯 필터 옵션:** 수집된 데이터 중 특정 장르만 필터링하거나 최소 동시 접속자 수(슬라이더) 설정
* **Main Page (메인 화면):**
  * **Header:** 로블록스 실시간 대시보드 타이틀 및 현재 전체 동시 접속자 총합 메트릭(`st.metric`)
  * **Body (실시간 차트 & 테이블):**
    * 사용자가 새로고침을 누를 때마다 크롤링된 최신 동접자 순위 Plotly 차트
    * `st.dataframe`을 이용해 현재 활성화된 로블록스 톱 게임들의 상세 목록 출력

---

실시간 크롤링 함수 구현: BeautifulSoup4와 requests를 사용하여 로블록스 인기 게임 정보(게임명, 동시접속자 수, 장르, 만족도)를 긁어오는 구조를 만들어 줘. 타겟 사이트 크롤링 시 차단을 방지하기 위해 헤더(User-Agent)를 반드시 설정해 줘.

네트워크 에러 방지 방어 코드: 만약 대상 웹사이트 구조 변경이나 네트워크 문제로 크롤링이 실패하면(try-except), 미리 준비된 10개의 최신 로블록스 게임 데이터셋(Mock Data)을 대신 반환하여 대시보드가 끊김 없이 작동하도록 예외 처리를 완벽하게 해줘.

성능 최적화: 스트림릿의 @st.cache_data(ttl=600)를 활용하여 데이터 수집 함수를 10분간 캐싱해 줘. 서버 부하를 줄이되, 사이드바의 '🔄 실시간 데이터 새로고침' 버튼을 누르면 st.cache_data.clear()가 실행되면서 강제로 데이터를 다시 긁어오게 매핑해 줘.

UI 및 시각화: 현재 수집 완료된 실시간 타임스탬프를 화면에 명시하고, Plotly 실시간 바 차트와 필터링이 가능한 데이터프레임을 직관적으로 배치해 줘."
